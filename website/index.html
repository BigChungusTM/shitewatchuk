Real-time monitoring of UK sewage discharges

About This Project

ShitWatch UK monitors real-time storm overflow events across England using data from the Environment Agency's Event Duration Monitoring (EDM) API. We track sewage discharges that last 10+ hours and publish detailed information every 90 minutes.

Data Sources

  • Environment Agency Real-Time API: Storm Overflow Event Data
  • Historical Flow Rates: Permit database analysis for volume estimates
  • Site Locations: GPS coordinates from EA discharge consents

How It Works

  1. Continuous Monitoring: We check the EA API every 5 minutes for ongoing discharge events
  2. Duration Tracking: Events are tracked from start to finish, with duration calculated in real-time
  3. 10+ Hour Threshold: Only significant events (10+ hours) are published
  4. 90-Minute Cycles: Every 90 minutes, we publish the latest data and post to Twitter
  5. AI Summaries: Each cycle includes an AI-generated summary explaining what the data means

What You'll Find Here

Each 90-minute cycle shows detailed information about sewage discharge events:

  • Water company responsible
  • Discharge site ID and location (GPS coordinates)
  • Receiving watercourse (river, stream, etc.)
  • Start time, end time, and total duration
  • Google Maps links to see exact locations
  • AI-generated summary of impacts and context
